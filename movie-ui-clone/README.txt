This ZIP contains:
- index.html : sample HTML layout (uses styles.css)
- styles.scss : SCSS single-file stylesheet to reproduce the provided design.
- Note: Browsers don't read SCSS directly. To preview in a browser:
  1) Use a Sass compiler (dart-sass) to compile styles.scss -> styles.css:
     sass styles.scss styles.css
  2) Open index.html in your browser.
